Starstream (prototype)

WebGL multiplayer space shooter +programmable ships

What is it?

Space flight & fight simulation

Navigate and explore space. Blow up stuff. Collect scrap. Build stuff.

Where can I play it?

Unfortunately, there is not even an alpha version avaialable yet. You may peek at your own risk at the development progress here: http://spacetit.heroku.com

Physics

Crash into stuff, or smash things together. Simulation runs both on client and server!

Web based

Runs in your browser with WebGL. Has cool stuff like shaders.

Multiplayer

Multiplayer by default, with websockets. You will be able to set up your own local servers.

Programmable ship

Ships expose an API to be programmed in JavaScript. Code runs on the server, safely, in it's own process. Github gist and/or repo integration planned.

Example API features:

  • Thrusters' force
  • Sensors
    • Accelerometers
    • Radar
    • Light
    • Radiation
    • Asteroid element component scanner
  • Communication
  • Asteroid mining equipment operation
  • Weapons and shields
